# CPRD-Aurum-UK-Veteran-codelist
SNOMED CT code list for identifying UK military veterans in CPRD primary care data

##  Manuscript title
Leveraging primary care data to understand military veteran health in England: Feasability study and matched-control comparative health profile

##  Dataset
CPRD Aurum

## Search strategy
We reviewed common terms reported in the literature and used a list of medical codes to identify veterans in secondary mental healthcare records (Leightley et al., 2023, Healthcare; Mark et al., 2020, Healthcare). We included terms relating to both regular and reserve Armed Forces service. 

We searched CPRD’s medical dictionaries for the following terms relating to military and veteran status: 
*military*
*veteran*
*armed forces*
*combat*
*army*
*air force*
*raf*
*navy*
*marine*
*royal engineer*
*corps*

The terms *reserve*, *deploy, *service person*, *serving*, *soldier* returned no results or resulted in codes already defined using the terms listed first and were therefore excluded from the final search strategy. 

Medical codes were cross-checked against military and veteran-related terms in the SNOMED browser to assess completeness. 
A group of veteran health experts, which included clinicians with experience supporting veterans in the NHS, reviewed the resulting code list to ensure it accurately reflected veteran status. 

This codelist inlcudes only clinical codes and does not include any patient-level data.
